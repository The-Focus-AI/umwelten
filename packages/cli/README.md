# @umwelten/cli

Command-line interface for umwelten.

## Install

```bash
pnpm add -g @umwelten/cli
# or
npx umwelten
```

## Commands

```bash
umwelten models          # List/search models
umwelten run             # One-shot prompt
umwelten chat            # Interactive chat
umwelten eval            # Run evaluations
umwelten sessions        # Session management
umwelten habitat         # Agent container (REPL, telegram, discord, web, serve, gaia)
umwelten mcp             # MCP client (chat, connect, test-tool)
umwelten browse          # Session browser TUI
```
