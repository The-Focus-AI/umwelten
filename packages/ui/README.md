# @umwelten/ui

User interface adapters for umwelten — Telegram, Discord, TUI, and CLI.

## Install

```bash
pnpm add @umwelten/ui
```

## What's inside

- **telegram/** — `TelegramAdapter` (Grammy-based Telegram bot)
- **discord/** — `DiscordAdapter` (Discord.js bot with channel routing)
- **tui/** — React Ink TUI: session browser, introspection browser, chat
- **cli/** — `CLIInterface`, `CommandRegistry`, `DefaultCommands`
